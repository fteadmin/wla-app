# WLA - World Lowrider Association

World Lowrider Association — uniting lowrider culture worldwide.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

```sh
# Clone the repository
git clone https://github.com/fteadmin/wla-app.git

# Navigate to the project directory
cd wla-app

# Install dependencies
npm install

# Start the development server
npm run dev
```
